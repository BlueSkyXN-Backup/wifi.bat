# wifi.bat
FIX Wifi AutoConfig BUG

通过禁用WIFI自动搜索来解决WIN在使用WIFI/WLAN无线网络时出现的周期性、有一定规律、阶段性的跳ping丢包抖动，在局域网内就开始出现这个情况，而网线直连没有问题
